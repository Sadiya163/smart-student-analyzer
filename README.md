## Project Description
Smart Student Analyzer is a Python-based project that evaluates student performance
by calculating average marks, grades, pass/fail status, and identifying weak subjects.

## Features
- Subject-wise marks input
- Average calculation
- Grade assignment
- Pass/Fail evaluation
- Weak subject detection

## Concepts Used
- Functions
- Dictionaries
- Loops
- Conditional statements

## How to Run
```bash
python smart_student_analyzer.py
